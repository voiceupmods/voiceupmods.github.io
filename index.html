Common Questions

How do I open the menu?

By default, the configuration key is 'V'. You can easily change this in your Minecraft control settings.

Is it required on the server?

Yes, for proximity voice chat to work, the mod must be installed on both the client and the server (Fabric).

Does it impact performance?

Not at all! VoiceUp is built to be ultra-lightweight, using parallel audio processing that doesn't affect your game FPS.

Support for Forge?

Currently, this version is exclusive to Fabric 1.21.1. A Forge port is in our future roadmap.
